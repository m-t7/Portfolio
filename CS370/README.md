### What code were you given? What code did you create yourself?
I was given starter code for the pirate intelligent agent and the Q-Training algorithm, and I created the code to implement the deep Q-learning algorithm and improve the agent’s pathfinding ability.

### What do computer scientists do and why does it matter?
Computer scientists design algorithms and systems to solve complex problems, enabling technological advancements that impact various industries and daily life.

### How do I approach a problem as a computer scientist?
I break down the problem into smaller tasks, identify patterns, create algorithms, and iterate on solutions based on feedback and testing.

### What are my ethical responsibilities to the end user and the organization?
Ethical responsibilities to the end user include ensuring privacy, security, fairness, and transparency in the code.
